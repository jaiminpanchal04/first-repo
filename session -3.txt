4.What do these HTTP status codes mean: 200, 301, 404, 500, 403?
ANS = 
 A explain the HTTP status code
 1.200 = 200 (OK) : the standard response for a successful HTTP request.
 2.301 = 301 (moved permanently) : The resource has a new permanent URL.
 3.404 = 404 (Not Found) : The server cannot find the requested resource or matching URL.
 4.500 = 500 (internal server error) : A generic error occurred on the server side.
 5.403 = 403 (Forbidden) : The server understands who you are, but you do not have permission.
