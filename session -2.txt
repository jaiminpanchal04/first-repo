1.What are the three main layers of software architecture? Write one line about each.?
ANS =  
 A three main layers of software architecture is User Interface, Business Logic layer, Database layer
 1.User interface layer = the user interface layer is the top layer that provides a user interface and handles user input/output
 2.Busines logic layer = The business layer contains the core logic that processes input, enforces rules and communication the UI and data layers.
 3.Database layer = The database layer is responsible for storing, retrieving, updating and deleting data in database or storage systems.
